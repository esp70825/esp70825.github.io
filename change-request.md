 # Exercise 06- The Mini Change Request
**Project Name:** Personal Introduction Project Webpage  
**Requested By:** Instructor 

---

## Proposed Change
The proposed change is to add an embedded Google Map feature that will show my hometown. It will require the following steps:
- Configure map to display hometown
- Write script to properly add an embed code through Google Maps into the HTML file
- Properly style the added map by changing the CSS accordingly

---

## Reasons for the Change
The instructor has requested this feature to improve the visual appeal of the webpage.
Instead of simply writing my hometown, the map will tie myself to a more visual location. This could have been added as an image, but the interactive element can make it more personal as well to the person visiting the page. 

---

## How This Change Will Add Value
One of the benefits of adding this element is that it will increase the visual appeal of the site, making it feel more elevated from just an ordinary page. It will also provide an instant point of information, rather than having to look up, it will be automatically on the page and the user can directly interact with the map.

---

## How This Change Will Affect the Constraints

#### Time
This addition will take 60-90 minuntes. This time will be allocated to learning the proper method to add the element, debug if necessary, make sure it works properly, it is integrated adequately with the page and styled correctly.

#### Quality
This will improve the quality of the site however, with writing and implementing more script, we can sometimes run  into slower loading speeds, although with a simple site, there should be minimal to no issues. 

#### Budget 
The cost of this addition will be zero, the necessary resources are free to use online and provided by Google.
